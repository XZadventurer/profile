# 简历智能顾问

一个基于AI的简历分析工具，帮助求职者优化简历内容。

## 产品说明

### 核心功能
- **本地文件上传**：支持TXT、MD、PDF格式简历上传，纯前端预览，保护隐私
- **AI智能分析**：使用Claude API进行专业简历分析
- **综合评分**：给出0-100分的简历评分，并用可视化圆环展示
- **优点提取**：识别简历中的亮点和优势
- **改进建议**：提供具体的优化方向
- **技能提取**：自动识别关键技能并标签化展示
- **职位推荐**：根据简历内容推荐适合的职位方向

### 解决的问题
1. 求职者不知道自己简历的优劣势
2. 缺乏专业的简历优化建议
3. 不清楚自己适合什么职位方向
4. 需要快速获得简历反馈

## 技术栈
- **前端**：React 18 + Vite
- **样式**：自定义CSS
- **PDF解析**：pdfjs-dist
- **后端代理**：Express + Node.js
- **AI服务**：Claude API

## 本地运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动后端代理服务器

在项目根目录下，打开第一个终端窗口：

```bash
node server.js
```

服务器将运行在 `http://localhost:3001`

### 3. 启动前端开发服务器

打开第二个终端窗口：

```bash
npm run dev
```

前端将运行在 `http://localhost:5173` 或 `http://localhost:5174`

### 4. 访问应用

在浏览器中打开前端地址，即可使用应用。

## 使用方法

1. 确保后端代理服务器和前端开发服务器都在运行
2. 点击上传区域选择简历文件（支持TXT、MD、PDF格式）
3. 查看简历预览内容
4. 点击"开始AI分析"按钮
5. 等待几秒后查看分析结果

## 注意事项

- 需要同时运行后端代理服务器（3001端口）和前端开发服务器（5173/5174端口）
- PDF文件解析依赖网络加载PDF.js worker
- AI分析需要有效的Claude API配置

## 特色

- 纯前端文件读取，保护用户隐私
- 响应式设计，支持移动端
- 美观的UI界面和动画效果
- 实时加载状态反馈

## 开发说明

如需修改API配置，请编辑 `server.js` 文件中的相关参数。

所有样式都在 `src/App.css` 中，可以根据需要自由修改。
